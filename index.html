View on GitHub

Institute for Software Integrated Systems, Vanderbilt University

Researching Mixture-of-Experts (MoE) architectures for robust, learning-based control methods in safety-critical systems. Implemented formal neural network verification using alpha-beta-CROWN (VNN-COMP 2021-2024 winner) to provide provable robustness guarantees for both MoE routers and expert models.
